# Explicit Intent Android

Project Android sederhana untuk mempelajari penggunaan explicit intent dalam perpindahan antar-activity pada aplikasi Android.

## ✨ Features

* Navigasi antar activity
* Penggunaan explicit intent
* Pengiriman data antar halaman
* Tampilan Android sederhana
* Struktur dasar aplikasi Android

## 🛠 Tech Stack

* Kotlin
* Android Studio
* XML Layout

## 🚀 Installation

1. Clone repository

```bash
git clone https://github.com/jaqckline/explicit-intent-android.git
```

2. Open project in Android Studio

3. Sync Gradle

4. Run application on emulator or Android device

## 📌 Purpose

Project ini dikembangkan untuk mempelajari konsep dasar explicit intent dan navigasi activity dalam pengembangan aplikasi Android menggunakan Kotlin.

## 📸 Preview

Project menampilkan:

* Halaman utama aplikasi

* Tampilan UI Android sederhana
